Design a complete web UI/UX mockup for an EdTech platform focused on sharing Chemistry learning materials, advanced Chemistry resources, community quizzes, and peer-to-peer question support.

Project context:
This is a community-based learning platform for students who study Chemistry, especially advanced Chemistry and specialized Chemistry topics. The platform allows admins to upload and organize study documents, create public quizzes, manage a community Q&A system, and track engagement. Users can log in with phone number + OTP, search and preview/download learning materials by tags, participate in quizzes, view leaderboards, and ask the community for help with difficult exercises.

Main goals:
- Build a clean, modern, academic but engaging education platform
- Focus on Chemistry and advanced Chemistry learners
- Encourage document discovery, quiz participation, and community interaction
- Create separate experiences for Admin and User
- The UI should feel practical, realistic, and ready to become a real product

Style direction:
- Modern, clean, minimal but lively EdTech design
- Academic and trustworthy, but still youthful and motivating
- Visual inspiration from science/chemistry: molecule icons, lab symbols, subtle chemistry patterns, formula-inspired decorations
- Primary colors: blue, white, light purple
- Accent colors: green or orange for CTA and reward/highlight areas
- Rounded cards, soft shadows, clear hierarchy, spacious layout
- Responsive web design, desktop-first mockup but also suitable for mobile adaptation
- Use a consistent design system with buttons, tags, badges, tables, cards, modals, and form components

Target users:
1. Admin:
- Upload documents for sharing
- Assign tags/categories to documents
- Create community quizzes, including quizzes with rewards
- Review and approve community questions before publishing
- Manage tags, users, leaderboard, and platform content

2. End users:
- Log in with phone number and OTP
- Search and filter Chemistry documents by tag/category
- Preview and download documents
- Join public quizzes
- View leaderboard/rankings
- Ask the community for help with exercises
- Upload images when posting a question
- Receive answers from the community after admin approval

Core features:
1. Document sharing
- Upload Chemistry learning materials
- Organize by tags, topic, grade, difficulty, category
- Search by keyword
- Filter by tags
- Preview documents before downloading
- Download documents
- Show popular, newest, and featured documents

2. Community quiz
- Public quizzes for all users
- Quick quiz creation by admin
- Optional reward-based quizzes
- Show number of questions, time limit, difficulty, participants
- Quiz result page with score and explanations
- Leaderboard integration

3. Community question support
- Users can post Chemistry questions when they do not know how to solve an exercise
- Users can upload images of the problem, handwritten work, formulas, or screenshots
- Posts must be approved by admin before being public
- Community can answer and vote useful responses
- Support comments and answer threads

4. Leaderboard and engagement
- Show top users based on quiz results, activity, or points
- Weekly, monthly, all-time ranking
- Gamified but still educational
